# Aqua Guide Presentation Plan

This repo ships with a ready demo flow and screenshot set for a 2-minute hackathon video or live presentation.

## Recommended flow

1. Start on the default safe homepage.
2. Search `Suffolk County, NY` to show the caution-state update.
3. Toggle `Quick Read` to demonstrate accessibility and clarity.
4. Save the location to show repeat-use value.
5. Ask the assistant a practical household question.
6. Open the boil-water action card for a concrete next step.
7. Use the location shortcut to jump to the advisory scenario.

## Talking points

- Broad problem: people do not want raw reports, they want clear guidance.
- Strong UX: the product stays calm, readable, and actionable.
- Honest AI story: the assistant can be powered by `AWS Bedrock or similar`.
- Presentation fit: every major interaction is visible in seconds.
